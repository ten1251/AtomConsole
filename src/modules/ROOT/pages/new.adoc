== Введение в возможности AsciiDoc

Это демонстрационная страница, созданная для того, чтобы показать богатые возможности языка разметки AsciiDoc. Здесь вы найдете примеры форматирования текста, вставки кода, изображений, видео, диаграмм и таблиц.

== Форматирование текста и заголовки

Заголовки разных уровней были продемонстрированы выше. Теперь покажем форматирование внутри абзаца.

* **Жирный текст** обрамляется звездочками.
* _Курсив_ обрамляется символами подчеркивания.
* `Моноширинный текст` для имен файлов и функций.
* [.underline]#Подчеркнутый текст# достигается с помощью роли (в некоторых темах может не работать, но в стандартном HTML5 работает через inline-роль).
* [.line-through]#Перечеркнутый текст# также делается через роль.
* Также можно использовать  специальные символы: (C) (R) и стрелки -> <-

== Вставка исходного кода (Java)

Ниже представлен пример простого Java-класса с нумерацией строк.

[source,java, linenums]
----
public class HelloWorld {
    public static void main(String[] args) {
        // Вывод сообщения в консоль
        System.out.println("Hello, AsciiDoc!");
    }
}
----

== Диаграммы PlantUML

AsciiDoc отлично интегрируется с PlantUML. 

[plantuml, my-schema, svg]
....
include::example$plant.puml[]
....

Диаграмма наглядно показывает жизненный цикл заявки на модерацию.

== Мультимедиа

AsciiDoc позволяет встраивать как изображения, так и видео.

### Вставка GIF-анимации

Предположим, у вас есть файл `animation.gif` в папке `images`.

image::animation.gif[Пример анимации, 400, 300]

### Встраивание видео

Видео с внешних хостингов можно встраивать с помощью макроса `video::`. Для RuTube и других платформ используется iframe.

++++
<iframe width="560" height="315" src="https://rutube.ru/play/embed/47e7ba138567a735a34e506f53ed0fcc" frameBorder="0" allow="clipboard-write; autoplay" webkitAllowFullScreen mozallowfullscreen allowFullScreen></iframe>
++++

Видео выше демонстрирует пример контента с RuTube.

== Сложная таблица

Таблицы в AsciiDoc очень гибкие. Ниже пример таблицы с объединением ячеек и выравниванием.

[cols="2,1,2", options="header", width="100%"]
|===
| Имя | Возраст | Хобби
| Алиса | 25 | Чтение книг
| Боб | 30 | 3+ *главных хобби*:
a| * Программирование
* Путешествия
* Фотография
| Чарли | 28 | Игра на гитаре
2.2+| Здесь текст занимает 2 строки и 2 колонки
|===
